# Jamnodes
A quick and easy Blender node setup for Animal Jam: Play Wild! models.

How to Install:

Place downloaded file in any folder
Open blender in your preferred project, go to File, and select Append
Locate the downloaded .blend file
Click into the NodeTree folder
Select all of the NodeTree contents, enable Fake User
Append
In the shader editor, using the node search, type Animal Jam
The nodes will show up, simply plug them in
You're done!
